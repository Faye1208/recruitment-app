 ### 技术栈

 + 前端：react-create-app + react16 + redux + react-router 4 + axios + antd-mobile 2

 + 后台：express + mongodb

 ---


 ### 运行项目

  + 获取项目:

        git clone git@github.com:Faye1208/recruitment-app.git

  + 打开项目所在目录:

        cd recruitment-app/

  + 安装项目依赖：

        npm install

  + 启动后台服务器

    node
